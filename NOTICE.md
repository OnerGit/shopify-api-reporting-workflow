# Notice

This repository is a public portfolio case study for a Shopify-style API reporting workflow.

It is not a runnable implementation repository. It intentionally excludes source code, tests, scripts, dependency files, Docker configuration, production connector logic, credentials, store domains, private paths, complete mock datasets, and client data.

The included sample outputs are small sanitized previews. They are intended to demonstrate reporting shape and handoff expectations, not to represent a full dataset or production export.

Any real client implementation would require store-specific Shopify Admin API configuration, approved access scopes, secure credential handling, cursor pagination, validation rules, privacy review, and field mapping adapted to the client's reporting definitions.
