# v0.4 Security Boundary Summary

The v0.4 connector template remains private. Public materials may mention the gated template at a high level, but must not include source code, detailed request or response material, store identifiers, private auth material, app identity values, or automation-token details.

Any public description should stay at the architecture and risk-control level: gated validation, redaction expectations, and client-specific adaptation requirements. It should not provide reusable connector implementation detail.
