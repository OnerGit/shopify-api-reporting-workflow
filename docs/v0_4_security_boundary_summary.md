# v0.4 Security Boundary Summary

The v0.4 connector template remains private. This public summary focuses on the security boundary around that work: gated validation, redaction expectations, and careful separation between portfolio evidence and client-specific implementation details.

Implementation details are maintained privately, including connector source code, detailed request and response handling, store identifiers, auth material, app identity values, and automation-token handling. The public case study describes the risk-control approach at a high level.
