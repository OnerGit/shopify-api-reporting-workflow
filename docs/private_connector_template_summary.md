# Private Connector Template Summary

v0.4 adds a private, mock-first connector template with environment-based configuration, a manually gated live validation path, redaction helpers, and sanitized validation summary guidance.

It is not a public production connector, not a public Shopify app, and not ready for all stores without client-specific adaptation.

This public summary must not be expanded into source code, configuration examples, request templates, response examples, app identity values, credentials, or store-specific setup steps.
