# Development Store Validation Summary

v0.3 adds private development-store validation notes and sanitized evidence guidance. The runnable workflows remain local mock demos only: REST-style fixtures for v0.1 and GraphQL-shaped fixtures for v0.2.

It does not include completed live or development-store validation evidence unless sanitized evidence is provided manually.

No real Shopify API calls, OAuth, tokens, store domains, customer data, SDKs, or production connector code are included.

The public repo should describe validation readiness and evidence-handling rules, not publish raw validation output.
